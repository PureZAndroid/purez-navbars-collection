Requirements:
- Android 7.0 and Up
- Substratum/OMS/RRO Support ROM
- Substratum Theme Manager (Get on Google Playstore)

How To Install:
- Download PureZ NavBars Theme
- Open Substratum Theme Manager
- Select Overlays
- Apply
- Done
